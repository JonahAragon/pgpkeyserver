--- layout: default title: OpenPGP Public Key Server ---

This server hosts and relays PGP keys via the SKS pool network. You may connect to this network by sending or receiving keys to pool.sks-keyserver.net (hkp).

This web page is provided as a convenient alternative to the usual methods of interacting with a keyserver. Key submission and retrieval are usually handled by key management software (such as GnuPG) on your local system.

This keyserver supports both the traditional email interface (for key submission only) PKS, and the more common HKP interface, for interaction with PGP key management software. The HKP interface allows direct interaction with the keyserver (perhaps via a web proxy server), while the email interface requires that the messages be queued and routed through the Internet via SMTP relays.
